# Assets 目录

此目录用于存放 VirtualDesktopAccessor.dll。

## 下载 VirtualDesktopAccessor.dll

1. 访问 [VirtualDesktopAccessor Releases](https://github.com/Ciantic/VirtualDesktopAccessor/releases)

2. 下载最新版本的 `VirtualDesktopAccessor.dll`
   - **重要**：选择与你的系统架构匹配的版本（通常是 x64）
   - **重要**：确保 DLL 版本与你的 Windows 版本兼容

3. 将下载的 `VirtualDesktopAccessor.dll` 放入此目录

## 兼容性说明

VirtualDesktopAccessor.dll 依赖于 Windows 的内部 API，不同 Windows 版本可能需要不同版本的 DLL：

| Windows 版本 | 说明 |
|-------------|------|
| Windows 11 23H2+ | 使用最新版本 |
| Windows 11 21H2/22H2 | 可能需要特定版本 |
| Windows 10 | 可能需要旧版本 |

如果遇到问题，请查看 [VirtualDesktopAccessor Issues](https://github.com/Ciantic/VirtualDesktopAccessor/issues) 获取帮助。
